# TCSS491-Between-Worlds
A short team-based game project for TCSS 491 where players navigate levels, avoid threats, and use simple interactions to reach the objective.

## Timeline

- **Week 3**
  - Get the structure of the game started, giving us a good framework to build on.

- **Week 4 (Prototype)**
  - Add character with basic map and movement.

- **Week 5**
  - Work on the functionality of the character, the map interactions, and the enemies.

- **Week 6**
  - Add levels and make combat with enemies work properly and also get to a good point with our switching dimensions feature.

- **Week 7 (Minimal Deliverable)**
  - Get to a point where we can play through a demo with combat and movement down to almost exactly what we want.

- **Week 8**
  - Add a lot of enemy maps and fully figure out character abilities.

- **Week 9**
  - Polish all functions of the game, including combat, movement, sound, art, etc.

- **Week 10**
  - Add as much cool music and art as we can to make the game look really pretty.
